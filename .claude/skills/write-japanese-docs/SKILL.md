# Write Japanese Docs

## 言葉遣い

- 意味範囲の広い動詞は避け、具体的な動詞を使用してください

  - 動作を行う → 動作する
  - ツールを用いる → ツールを使う
  - 処理を行う → 処理する

- 冗長な表現を避け、簡潔に記述してください

  - 実行することができる → 実行できる
  - 可能であるということ → 可能であること
  - 劣化していることにより → 劣化したため
  - ～している状態である → ～している

- 形式名詞や不要な補助表現の使用を抑制してください

  - ～ということ → 省略または名詞化
  - ～という形で → ～として
  - ～のほう → 削除

- 可能表現は簡潔な形に統一してください

  - ～することが可能 → ～できる
  - ～が可能となっている → ～できる

- 曖昧な副詞・表現を避けてください

  - 適切に
  - ある程度
  - いろいろな
  - 必要に応じて

- 口語的な表現は使用しないでください
  - とりあえず
  - なんか
  - ～みたいな
